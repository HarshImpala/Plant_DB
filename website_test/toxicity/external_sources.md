# External Toxicity Sources

Use these sources to review toxicity for humans, cats, and dogs.

## Veterinary / Pet Toxicity

1. ASPCA Animal Poison Control - Toxic and Non-Toxic Plants  
   https://www.aspca.org/pet-care/animal-poison-control/toxic-and-non-toxic-plants
2. Pet Poison Helpline - Toxic Plants  
   https://www.petpoisonhelpline.com/poisons/plants/
3. Merck Veterinary Manual - Poisonous Plants  
   https://www.merckvetmanual.com/toxicology/plant-poisoning
4. VCA Animal Hospitals - Toxic Plant Guides  
   https://vcahospitals.com/know-your-pet
5. UC Davis VetMed - Toxic plants and pet safety resources  
   https://www.vetmed.ucdavis.edu/

## Human Toxicity

1. U.S. FDA - Poisonous Plant Database  
   https://www.accessdata.fda.gov/scripts/plantox/
2. Poison Control (U.S.) - Plant Poisoning Information  
   https://www.poison.org/articles/plant
3. NCBI Bookshelf / toxic plant references (search entry point)  
   https://www.ncbi.nlm.nih.gov/books/

## Regional / Botanic Reference

1. North Carolina Extension Gardener Plant Toolbox  
   https://plants.ces.ncsu.edu/
2. Royal Horticultural Society (RHS) plant safety guidance  
   https://www.rhs.org.uk/plants/types/houseplants/poisonous-house-plants

## Notes

- Treat this as evidence gathering, not medical/veterinary diagnosis.
- Prefer sources with explicit species-level statements.
- If sources conflict, keep the more conservative status (`toxic` > `possibly_toxic` > `unknown` > `not_toxic`) and record the conflict in notes.
